# Security Policy

## Supported Versions

This project currently supports the latest `master` branch for security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue.

Use GitHub Security Advisories for private reporting:

- https://github.com/JasonTM17/shopapp_frontend/security/advisories/new

Please include:

- A clear description of the issue
- Reproduction steps or proof of concept
- Potential impact assessment
- Suggested mitigation (if available)

We will acknowledge reports and investigate as quickly as possible.
